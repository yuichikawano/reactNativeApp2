# AuthAppの概要

#### create project
```zsh
npx create-expo-app modal-app --template https://github.com/kadikraman/expo-starter

```
#### install
```zsh
npm i
```

#### 起動方法
```zsh
npm run start
```
#### sitemapの表示
```zsh
 npx expo-router-sitemap
```
#### chace clear start
```zsh
npm expo start --reset-cache
```
